Chat App

python Full Stack

Nov 20

I developed a real-time chat application using Django for the backend and a responsive frontend built with HTML, CSS, and JavaScript.
The app supports user authentication, one-to-one chats, group conversations, and real-time messaging using WebSockets (Django Channels).
It stores chat history securely in the database and provides a smooth, mobile-friendly interface.
This project helped me understand real-time communication, asynchronous processing, and building clean, user-focused UI.
